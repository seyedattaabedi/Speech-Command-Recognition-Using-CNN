# Speech Command Recognition Using CNN

This project implements a Convolutional Neural Network (CNN) for speech command recognition using TensorFlow/Keras.

The model classifies spoken commands by converting raw audio signals into spectrograms and learning discriminative audio features.

## Dataset

Google Speech Commands Dataset

## Technologies

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Seaborn
* PyAudio

## Features

* Audio preprocessing
* Waveform visualization
* Spectrogram generation (STFT)
* CNN-based audio classification
* Early stopping
* Confusion matrix evaluation
* Prediction on unseen audio files
* Real-time microphone recording
* Model weight saving and loading

## Workflow

1. Load speech dataset
2. Convert waveform to spectrogram
3. Normalize inputs
4. Train CNN model
5. Evaluate performance
6. Predict spoken commands
7. Test with microphone input

## Goal

The objective of this project is to build a speech recognition pipeline using deep learning techniques and gain practical experience in audio processing and speech classification.
